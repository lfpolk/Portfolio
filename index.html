NHL Game Predictor
Full stack web app using Python Flask, SQLAlchemy, and SQLite on the backend and HTML/CSS/JavaScript on the front end. Implements the Beautiful Soup framework to parse HTML files of advanced statistics from hockey-reference.com to simulate NHL games. Deployed to Heroku.


In Your Bag - Disc Golf Bag Visualizer
Full stack web app using PostgreSQL, Express, React, and Node. Users can use this app to visualize the discs in their bag to see what holes they may need to fill. Uses a JWT authentication and authorization system for users to securely create and edit their own bags.


University of AR: On campus assistant
Mobile app implemented in Swift which instanly returns the name and information about landmarks by pointing your camera at it. Trained a machine learning model using CreateML and a sample of over 1000 images to recognize over 30 different points of interest on the University of Arkansas campus. Uses CoreML to classify the image from the camera, and all the data from that classification is read from a JSON file and presented to the user
